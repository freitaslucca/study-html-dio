Princípios Básicos do HTML

HTML (HyperText Markup Language) é a linguagem padrão utilizada para criar páginas web. Ele permite que você estruture seu conteúdo e adicione vários tipos de elementos que compõem uma página da web.

Elementos de Cabeçalho

Elementos de cabeçalho variam de <h1> a <h6> e são usados para definir títulos e subtítulos.

Este é um título principal <h1>

Este é um subtítulo <h2>

Este é um subtítulo menor <h3>

Entradas de Formulário

Os elementos de entrada são usados em formulários para capturar dados do usuário.

Insira seu email

Insira sua senha

É muito importante que as informações estejam corretas, revise-as se necessário!

Listas

Listas podem ser ordenadas (<ol>) ou não ordenadas (<ul>).

Lista Ordenada

  1. Lucca
  2. Carlos
  3. Ricardo
  4. Richard

Links

Links são definidos com a tag <a>. Eles podem redirecionar para outra página ou para uma âncora na mesma página.

Link externo:

LinkedIn Lucca

Link interno:

Redirecionar para lembrete

Comentários

Comentários são usados para explicar partes do código e são ignorados pelo navegador. Eles são definidos assim:

<!-- Este é um comentário -->
